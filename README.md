# SQL Ecommerce Project

## Project Overview
This is a SQL Capstone Project for an Ecommerce Store (HimanshuShop). The project demonstrates database design, table relationships, foreign keys, sample data generation, and business analysis using SQL.

## SQL Skills Demonstrated
- Database Design
- Relational Database Modeling
- Primary & Foreign Keys
- Table Creation (DDL)
- Data Insertion (DML)
- Joins (INNER JOIN)
- Aggregate Functions (SUM, COUNT, AVG)
- GROUP BY & ORDER BY
- Filtering (WHERE)
- Business Analysis Queries
